Authorize & Setup

Import property definitions from cloudbeds. 

Synchronize rooms and common areas, personalize checkin and checkout times, code length sizes on devicethread

Sync Guest Reservation Info

devicethread receives guest and stay information from Cloudbeds like name, email, phone number, date, time, and confirmation number for both individual and group reservations

Sync Guest Access Key

devicethread sends Cloudbeds guest access code as well as real-time alerts to change the status of the stay to Check-in/ Out for guest rooms as well as common public areas. Codes could be 4-digit pin that could be random or personalized to the guest (e.g. last 4-digits of guest mobile number). Non-pinpad keys (NFC or BLE) are sent directly to the guests.

Cloudbeds can email, text or use Whistle app to share code or mobile key with the guest prior to arrival offering true contactless check-in.  

Sync Reservation lifecycle

Works with any changes to reservations or rooms.

Monitor Access

All guest and staff access logged for analysis and detecting anomalous behavior.

Staffs receive low battery alerts as well as malfunctioning locks or hubs for prompt attention.
